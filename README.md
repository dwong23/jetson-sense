# jetson-sense
Learning sandbox for deploying TensorRT accelerated deep learning inference networks on a NVIDIA Jetson TX2 with RGB and depth sensing from an Intel RealSense D415.

## Requirements
### Hardware
* [NVIDIA Jetson Embedded Platform: Nano/TX1/TX2/Xavier](https://www.nvidia.com/en-us/autonomous-machines/embedded-systems/)
* [Intel RealSense Depth Camera](https://www.intelrealsense.com/)

### Software
* [Python 3](https://docs.python-guide.org/starting/installation/)
* [jetson-inference](https://github.com/dusty-nv/jetson-inference)
* [Intel RealSense SDK](https://github.com/IntelRealSense/librealsense)
* [OpenCV](https://github.com/opencv/opencv)
